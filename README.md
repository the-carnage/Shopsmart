# ShopSmart
A full-stack app with React/Vite (client) and Express (server).

## Requirements
1. SQLite3 DB
2. Prisma ORM
3. CRUD API
4. Deploy (Render/Vercel or Docker)
5. Resolve CORS

## Usage
`chmod +x bash.sh`
- `./bash.sh install`: Install dependencies
- `./bash.sh dev`: Run locally
- `./bash.sh test`: Run tests
- `./bash.sh build`: Build frontend
- `./bash.sh docker`: Build Docker images
